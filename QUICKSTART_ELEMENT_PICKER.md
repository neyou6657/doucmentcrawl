# 🚀 元素选择器快速开始

只需 **3 步**，无需了解 CSS 选择器！

## 步骤 1️⃣ ：选择链接区域

```
1. 点击控制面板中的 "🎯 选择链接区域" 按钮
2. 移动鼠标到包含导航链接的区域（如侧边栏）
3. 看到橙色高亮后，点击该区域
4. 确认提示显示找到的链接数量
```

**效果：**
- 页面出现半透明遮罩
- 鼠标悬停的元素显示橙色边框
- 黑色提示框显示元素信息

## 步骤 2️⃣ ：选择内容区域

```
1. 点击 "🎯 选择内容区域" 按钮
2. 移动鼠标到包含文档内容的区域（如主内容区）
3. 点击该区域
4. 看到确认提示
```

## 步骤 3️⃣ ：开始工作

```
1. 点击 "1. 扫描链接" 按钮
2. 查看扫描结果
3. 点击 "2. 开始抓取" 按钮
4. 等待完成后导出数据
```

## 💡 快速提示

### 选错了怎么办？
- **按 ESC 键** 取消选择
- 或直接重新点击按钮覆盖旧选择

### 如何知道选对了？
- 查看提示框显示的选择器
- 查看链接数量是否合理
- 控制面板底部会显示当前选择器

### 最佳选择建议
- ✅ 选择**容器元素**（包含多个链接的区域）
- ✅ 选择**语义元素**（nav、aside、main、article）
- ❌ 避免选择**单个链接**
- ❌ 避免选择**动态元素**（广告、弹窗）

## 🔍 常见场景

### GitHub Wiki
```
链接区域：右侧目录 → .wiki-rightbar a
内容区域：文章内容 → #wiki-body
```

### MDN Web Docs
```
链接区域：左侧导航 → nav.sidebar a
内容区域：文章内容 → article#content
```

### React 官方文档
```
链接区域：左侧边栏 → .sidebar a
内容区域：主内容区 → article
```
