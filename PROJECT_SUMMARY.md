## 🎯 核心功能

### 1. 智能链接扫描
- 支持多种导航栏选择器
- 自动去重和URL标准化
- 基于URL路径自动分类

### 2. 内容提取
- **CSS选择器**：自动查找主内容区域
- **XPath支持**：精确定位特定元素
- **多元素提取**：
  - 标题结构（h1-h6）
  - 代码块（含语言识别）
  - 表格数据
  - 图片信息
  - 链接关系

### 3. 批量抓取
- 并发控制（可配置1-N个并发）
- 请求延迟（避免服务器压力）
- 实时进度显示
- 错误处理和日志

### 4. 多格式导出
- **JSON**：完整的结构化数据
- **Markdown**：易读的文档格式
- **CSV**：适合数据分析

### 5. 用户界面
- 可视化控制面板
- 实时进度条
- 统计信息展示
- 友好的操作提示

